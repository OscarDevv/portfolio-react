## Visualizando o projeto

Nesta seção, você poderá ver como acessar o portfólio tanto na web quanto na sua máquina.

### Na web

Esse é o mais fácil de todos, tudo o que você precisa é acessar [o link](https://oscardevv.vercel.app):

```txt
https://oscardevv.vercel.app
```

### Localmente

Para acessar localmente, ou seja, na sua máquina, você precisa ter o projeto instalado, então o primeiro comando a se fazer é clonar o projeto, então use este comando no terminal, caso você tenha o Git instalado:

```bash
git clone https://github.com/oscardevv/portfolio-react.git
```

Se não tiver o Git instalado, vá pro repositório e clique em Code, e nele baixe o arquivo ZIP no botão "Download ZIP", e depois abra ele.

---

Agora seu próximo passo é abrir o projeto e rodar mais alguns comandos no terminal, se não tiver um gerenciador de pacotes instalado, você não conseguirá prosseguir:

```bash
# Rode este comando:
npm install

# Se você não usa NPM, mas usa outro gerenciador de pacotes, rode um desses comandos que possam ser qual você usa:
pnpm install
yarn install
bun install
```

---

E por fim, rode o último comando no terminal:

```bash
# Rode este comando:
npm run dev

# Se você não usa NPM, mas usa outro gerenciador de pacotes, rode um desses comandos que possam ser qual você usa:
pnpm dev
yarn dev
bun run dev
```

Após isso, abrirá a visualização do projeto no seu navegador padrão em https://localhost:5173 (o número pode mudar).

## Contribuição

Caso você queira contribuir e melhorar este projeto, crie um fork e faça um pull request, também você pode adicionar uma descrição do que melhoraria com esse pull request, o que mudou, e aí eu vou analisar e decidir se aceito ou não.

**Passo a passo:**

1. Crie um fork
2. Crie uma nova branch (`git checkout -b feat/NovaFuncionalidade`)
3. Faça um commit com suas mudanças (`git commit -m "Nome do commit que adiciona funcionalidade"`)
4. Faça um push (`git push origin feat/NovaFuncionalidade`)
5. Abra uma Pull Request (PR)
